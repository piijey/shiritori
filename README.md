# しりとり
【作成中】しりとりぼっとのリポジトリです。

## インストール
```sh
npm install
```

[kuromoji.js](https://github.com/takuyaa/kuromoji.js) （日本語形態素解析器）を利用します。
npm でインストールした `kuromoji/dict/` 下のファイルを、 `public/` 下にディレクトリごとコピーします。
```sh
cp -r node_modules/kuromoji/dict public/kuromoji-dict
```
